# センタク — 雨の前に洗濯物を取り込むスコアアタックアプリ

## v3 (Latest) — 安定版

最新の安定バージョンです。本アプリの基本フロー・UI・スコアシステムがすべて実装されています。

---

## 概要

**ゲームルール:**
- 天気情報とリアルタイムに連動。雨が降る予定時刻をカウントダウン
- 雨まで60分を切ると「ウィンドウ解放」
- ユーザーは右スワイプで窓を開け、その直後に左スワイプで洗濯物を取り込む
- 雨が降る時刻に**近いほど高スコア** (S～Fランク)
- パートナー機能で複数人が同じ洗濯物の取り込みを競う

**スコアシステム:**
| 取り込みタイミング | スコア | ランク |
|---|---|---|
| 雨の5分以内 | 9999 | S |
| 雨の15分以内 | 8000~ | A |
| 雨の30分以内 | 6000~ | B |
| 雨の55分以内 | 3000~ | C |
| それ以上 | 500 | F |

## 画面レイアウト

### ヘッダー
- アプリタイトル「センタク」
- 右側: パートナー・履歴・追加のアイコンボタン (タップでモーダル展開)

### 情報バー
- 現在地: 東京・渋谷区
- 気温: 18°C
- **残り時間: 大きく表示** (カウントダウン)

### メイン領域 (フル画面)
1. **ウィンドウ解放前**: 灰色の盲が閉まった窓を表示
2. **ウィンドウ解放後**: 右スワイプで盲が右にスライド → 「窓オープン！」
3. **窓を開いた後**: 洗濯紐からカラフルなTシャツが出現
4. **取り込み中**: 左スワイプで1枚ずつTシャツが消える
5. **完了**: スコア表示 → 「もう一度プレイ」ボタン

### ステータスバー (下部)
- 現在のフェーズに応じたガイダンス表示

## フェーズ遷移

```
SAFE (安全)
  ↓ (60分前に到達)
WARNING (警告)
  ↓ (窓解放・ユーザーが操作)
WINDOW (ウィンドウ解放)
  ↓ (時間切れ)
RAINED (雨が降った)
```

各フェーズでは背景グラデーションが変わり、ユーザーに視覚的に時間の迫力を伝えます。

## 主要なコンポーネント

### `page.tsx` — メインロジック

```typescript
// フェーズ判定
function getPhase(remaining: number): Phase {
  if (remaining <= 0) return "rained"
  if (remaining <= WINDOW_MINUTES) return "window"  // 窓解放
  if (remaining <= WINDOW_MINUTES + 20) return "warning"
  return "safe"
}

// スコア計算
function calcScore(minutesBefore: number) {
  if (minutesBefore <= 5) return { score: 9999, rank: "S" }
  // ... A, B, C, F ランク
}
```

**状態管理 (`AppState`):**
- `windowOpen`: ウィンドウを開いたか
- `collected`: 洗濯物を取り込んだか
- `phase`: 現在のフェーズ
- `remainingMinutes`: 雨までの残り時間
- `score`, `rank`: 計算結果
- `partnerCollected`: パートナーが先に取り込んだか

### `WindowSwipe.tsx` — 窓UI

- 右ドラッグで盲がスライド
- 70% ドラッグしたら自動的に窓が「開く」
- マウス・タッチ両対応
- アニメーション: 盲の移動・回転・太陽の登場

### `TShirtSwipe.tsx` — 洗濯物UI

- 洗濯紐にクリップで止められたTシャツを表示
- 5枚のTシャツが用意されている（色分け）
- 左ドラッグで1枚ずつ取り込み
- 65% ドラッグしたら自動的にTシャツが「消える」
- 5枚すべて完了時に親に通知

### `ScoreDisplay.tsx` — スコア表示

- ランクバッジ (S/A/B/C/F) を円形で表示
- スコアをカウントアップアニメーション付きで表示
- スコアバーのプログレス表示
- 色分け: S=黄色, A=緑, B/C/F=青系

## 開発ガイド

### デバッグ用定数変更

```typescript
// app/page.tsx
const TOTAL_MINUTES = 70  // 本番環境では 180 (3時間)
const WINDOW_MINUTES = 60  // ウィンドウ解放タイミング
```

**時間スケール:** 1秒 = 1分 (デバッグ快適化)

### パートナー機能の試験

```typescript
// app/page.tsx の PARTNER オブジェクト
const PARTNER = {

```
