# Antigravity カスタムルール

このプロジェクトにおいて、AIアシスタント（Antigravity）が遵守すべき追加ルールです。

## 1. 言語設定
- **チャット応対**: すべての会話は日本語で行うこと。
- **アーティファクト/ドキュメント**: 以下のファイルを含む、ユーザーが確認するすべての文章は**必ず日本語**で作成すること。
    - `implementation_plan.md`（実装計画）
    - `task.md`（タスクリスト）
    - `walkthrough.md`（解説・ウォークスルー）
    - `analysis_results.md` 等の報告書

## 2. モバイル開発優先
- **ターゲット**: Web版（Next.js）よりも、モバイル版（Expo/React Native）の開発および修正を最優先すること。
- **操作感**: iPhone等の実機での操作性（スワイプの感触、ジェスチャの競合回避）を第一に考えること。

## 3. デザイン方針
- **質感**: 手書きイラスト風、ミニマル、高コントラストなデザインを基調とすること。
- **フォント**: 柔らかい印象を与えるため、丸文字（Rounded）系のシステムフォント設定を優先すること。
